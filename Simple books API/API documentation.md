# Simple Books API - Request Dependency Matrix

| Order | Request Name       | Depends On                    | Purpose                              | Notes                                   |
| ----- | ------------------ | ----------------------------- | ------------------------------------ | --------------------------------------- |
| 1     | Get Status         | None                          | Verify API availability              | Independent request                     |
| 2     | List Of Books      | None                          | Retrieve available books             | Independent request                     |
| 3     | Get Books          | None                          | Retrieve book information            | Independent request                     |
| 4     | Get Single Book    | None                          | Retrieve details of a specific book  | Independent request                     |
| 5     | New Authentication | None                          | Generate authentication token        | Required for protected endpoints        |
| 6     | New Order          | New Authentication            | Create a new book order              | Uses authentication token               |
| 7     | GET All Orders     | New Authentication, New Order | Retrieve all existing orders         | Requires authenticated access           |
| 8     | Get An Order       | New Order                     | Retrieve details of a specific order | Uses stored order ID                    |
| 9     | Update Orders      | Get An Order                  | Update an existing order             | Uses stored order ID                    |
| 10    | Delete Orders      | New Order / Update Orders     | Delete an existing order             | Uses stored order ID and existing order |

## Dependency Flow

```text
Get Status
     │
List Of Books
     │
Get Books
     │
Get Single Book

New Authentication
        │
        ▼
    New Order
        │
        ├──► GET All Orders
        │
        ├──► Get An Order
        │          │
        │          ▼
        │     Update Orders
        │          │
        ▼          ▼
       Delete Orders
```

## Dynamic Variables

| Variable             | Source Request     | Used By                                                               |
| -------------------- | ------------------ | --------------------------------------------------------------------- |
| Authentication Token | New Authentication | New Order, GET All Orders, Get An Order, Update Orders, Delete Orders |
| Order ID             | New Order          | Get An Order, Update Orders, Delete Orders                            |

## Notes

* Requests related to books and API status are independent and can be executed individually.
* Authenticated endpoints require a valid token generated by the **New Authentication** request.
* Order-related requests depend on the successful creation of an order.
* Dynamic values such as authentication tokens and order IDs are stored as collection variables and reused throughout the collection.
* The collection demonstrates request chaining and end-to-end API workflow validation.
